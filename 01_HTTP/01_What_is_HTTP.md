# What is **HTTP**
## HTTP란?
HTTP ( Hyper Text Transfer Protocol ) : <br>
**인터넷에서 데이터를 주고받을 수 있는 <u>프로토콜</u>**
> **프로토콜** : 규칙, 방식, 규격<br>

**핵심** : 서버의 역할은 요청에 대한 응답을 보내주는 것!

<br>

## 요청
> 클라이언트가 서버에게 해당 주소의 정보를 요청 <br>
-> 요청에 대한 정보를 담아 서버에게 전송

### **요청** 정보의 구조
- 시작줄 - 메서드, 주소, 버전
- 헤더 - 요청에 대한 정보
- 본문 - 데이터

<br>

## 응답
> 서버가 클라이언트에 요청에 따라 정보가 담긴 메시지, <br>
**HTTP 메시지**를 클라이언트에게 전송

### **응답** 정보의 구조
- 시작줄 - 버전, 상태 코드, 상태 메시지
- 헤더 - 응답에 대한 정보
- 본문 - 데이터

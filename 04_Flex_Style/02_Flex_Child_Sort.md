# Flex to Child Sort
## 자식 요소 정렬하기
### Justify-content
속성값 | 설명
:--- | :---
flex-start | 자식 요소를 시작 방향으로 정렬
flex-end | 자식 요소를 종료 방향으로 정렬
center | 자식 요소를 가운데로 정렬
space-between | 자식 요소들의 좌우 사이 여백만 균일하게 배분해서 정렬
space-around | 자식 요소들의 여백을 균일하게 배분해서 정렬
space-evenly | 자식 요소들의 여백을 양끝까지 균일하게 배분해서 정렬

# HTML Menu Tag
## \<ul> 태그
순서가 없는 메뉴를 목록으로 만들 때 사용하는 태그 <br>
자식태그로 \<li>를 사용
``` html
<ul>
    <li>목록1</li>
    <li>목록2</li>
    <li>목록3</li>
</ul>
```

<br>

## \<ol> 태그
순서가 중요할 때 사용하는 목록을 만들기 위해 사용하는 태그 <br>
자식 태그로 \<li>를 사용
``` html
<ol>
    <li>목록1</li>
    <li>목록2</li>
    <li>목록3</li>
</ol>
```

<br>

## \<dl>, \<dt>, \<dd> 태그
정의형 목록을 만들기 위해 사용하는 태그 <br>
자식 태그로 \<dt>, \<dd>를 사용
``` html
<dl>
    <dt>정의할 제목</dt>
    <dd>정의할 내용</dd>
</dl>
```
